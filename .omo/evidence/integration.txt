## Integration Summary
src/main.ts rewritten (41 → ~110 lines) to wire all 16 modules:
- WebGLRenderer with preserveDrawingBuffer:true (G17) + antialias
- createOutlineEffect(renderer) → effect.render(scene, camera) in loop
- createScene() → scene (default camera discarded)
- scene.add(planet) — spherical world
- scene.add(createRegions()) — 3 regions
- scene.add(createNpcs()) — 5 NPCs
- scene.add(createPlayer().group) — player
- createFollowCamera() → { camera, update } (replaces default camera)
- questManager singleton from ./quest
- createInteraction(player, npcs, questManager)
- createHud(questManager)
- createIntro(onStart) — onStart: hud.show() + player.setEnabled(true)
- installDebugApi(getState) — live state snapshot + teleport/completeQuest/pause/resume
- Main loop: clock.getDelta/elapsedTime, tickFps, conditional update (playing && !paused), effect.render
- Resize handler updates follow camera aspect + renderer size
- intro.show() at startup

## Debug API (window.__game, DEV only)
getState() returns:
- player: { position: [x,y,z], up: [x,y,z] }
- camera: { position: [x,y,z], up: [x,y,z] }
- scene, renderer (live refs)
- quests: questManager.getQuests()
- teleport: (x,y,z) => player.position.set(x,y,z)
- completeQuest: (id) => questManager.completeQuest(id)
- pause/resume: synthetic Escape keydown toggles HUD pause state
