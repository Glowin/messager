## 6. Design Decisions
- Styles injected via JS (createElement('style')) — NOT CSS import, because tsx cannot parse CSS imports (Task 8 gotcha). Self-contained module, tsx-compatible.
- typeof document !== 'undefined' guard on showOverlay — Node (tsx) skips DOM, state machine still works.
- completeQuest is public — debug API (Task 7) expects completeQuest(id) for QA shortcut.
- startQuest only works on locked quests (no-op on active/completed) — prevents accidental resets.
- advanceStep only works on active quests — no-op on locked/completed.
- AudioManager.playSfx('quest-complete') called in completeQuest — no-throw in Node (Task 4 verified).
