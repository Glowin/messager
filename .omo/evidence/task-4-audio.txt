## 4. Fault tolerance (npx tsx -e, no WebAudio in node)
```
playBgm: OK (no throw)
playSfx(jump): OK (no throw)
playSfx(quest-complete): OK
resume: OK (no throw)
Howler.ctx: undefined (node, no WebAudio)
```
- playBgm/playSfx do not throw even without WebAudio context.
- resume() guards `if (ctx && ctx.state === 'suspended')` — no throw when ctx undefined.
- onloaderror/onplayerror handlers swallow missing/corrupt audio errors.

## 6. src/audio.ts API
- AudioManager singleton (module-level instance export)
- playBgm(): lo-fi loop, volume 0.4, loop:true, no-throw, idempotent (no-op if already playing)
- playSfx(name): 'jump'|'footstep'|'quest-complete'|'dialogue', volume 0.6, lazy-cached
- setMuted(bool): calls Howler.mute(bool), tracks state in this.muted
- isMuted(): returns tracked mute state
- resume(): Howler.ctx?.resume() with suspended-state guard (for Task 15 Begin gesture unlock)
