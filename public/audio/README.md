# Audio Assets

All files in this directory are **CC0 / self-generated placeholders** synthesized via
`ffmpeg -f lavfi` (sine + tremolo + fade). They are unencumbered by copyright and may be
freely replaced with higher-quality CC0 assets.

## Placeholder Source
Generated programmatically with ffmpeg (no third-party audio sampled).
- `bgm.mp3` — 3s 220Hz tremolo tone (lo-fi ambient placeholder, loopable)
- `sfx/jump.mp3` — 0.15s 660Hz blip
- `sfx/footstep.mp3` — 0.08s 150Hz soft tap
- `sfx/quest-complete.mp3` — 0.4s 880Hz tremolo chime
- `sfx/dialogue.mp3` — 0.1s 500Hz click

## Recommended CC0 Replacements (freesound.org)
When replacing, use CC0-licensed assets from freesound.org (search filter: CC0):
- bgm: search "lo-fi loop" (CC0) — e.g. short ambient piano loop
- jump: search "jump" (CC0) — short upward blip
- footstep: search "footstep" (CC0) — soft single step
- quest-complete: search "chime" / "level up" (CC0)
- dialogue: search "blip" / "click" (CC0) — UI advance sound

Keep replacements short (< 5s for sfx, < 60s loopable for bgm) and verify CC0 license
on each asset's freesound page before committing.
